Part 1 — AI Graphics & Image Generation Tools

4 Tools
Midjourney
No Official API Paid Only
The gold standard for artistic image generation. Midjourney produces strikingly creative visuals — concept art, editorial illustrations, surreal scenes, product mockups — with a distinctive aesthetic that's hard to replicate. Available via Discord and the Midjourney web app.

High-quality artistic images, concept art, mood boards, editorial illustration, hero images
Limitations
  • No official API — third-party wrappers (MidAPI.ai, Apiframe) are unofficial and fragile
  • No reliable text rendering in images
  • Images are public by default on Basic/Standard plans
  • Discord-native UX; web app is newer and less feature-complete
API Access
❌ No official API. Third-party unofficial wrappers only — not recommended for production agent workflows.
DALL-E 3 / GPT-Image-1 (OpenAI)
✓ Full API Free via ChatGPT
OpenAI's image generation available via ChatGPT and the developer API. GPT-Image-1 (released April 2025) is natively multimodal and token-priced. Excels at instruction-following — you can describe exactly what you want and it delivers. Best API story of any image generator. Strong for programmatic workflows and agent integration.

Adobe Firefly
API (Enterprise) Free Tier
Adobe's generative AI platform for images, vectors, and video. Commercially safe — trained exclusively on Adobe Stock, licensed content, and public domain material. Deep integration with Photoshop (generative fill), Illustrator (vector generation), and the full Creative Cloud suite.

  • Trained on stock imagery — results can feel "stock-y" and less creatively adventurous
  • API is enterprise-tier, not self-serve like OpenAI

Part 2 — AI UI/UX Design Tools

4 Tools
Figma + Figma AI
File API ✓ Free Tier ⭐ Recommended
The industry-standard UI/UX design platform. Figma AI layers generative capabilities on top of an already world-class tool: text-to-prototype (Figma Make), AI component suggestions, design generation from prompts, Figma Buzz (on-brand asset scaling), Figma Sites (publish responsive websites), and Figma Draw (vector illustration). The tool every designer, PM, and developer already knows.

Galileo AI
No API Paid ⭐ Recommended

Galileo specializes in generating complete, beautiful UI designs from a description — not rough wireframes, but polished, production-quality mockups. Trained on thousands of exceptional UI designs. Exports directly to Figma for refinement. Generates both web and mobile UI.

Part 3 — AI General Design Tools

3 Tools
Canva AI (Magic Studio)
API (Enterprise) Free Tier
The dominant all-in-one visual design platform for non-designers. Magic Studio bundles AI image generation, Magic Resize (auto-adapt to any format), background removal, AI text generation, animated designs, and Magic Switch (format conversion). 190+ million monthly active users. The go-to tool for marketing materials, social media, presentations, and content.

Adobe Express
No Standalone API Free Tier
Adobe's lightweight template-based design tool for non-designers. Integrates with Creative Cloud assets and Firefly AI (generative fill, text effects, image generation, background removal). Great for social posts, short videos, flyers, and marketing assets — especially if already in the Adobe ecosystem.

🏆 Recommended Stack for Cricket

Best combination of quality, cost, and API access for the K2S0 design workflow.

Tool 1 · Primary Design Environment
Figma + Figma AI
UI/UX Design, Wireframing, Prototyping, Dev Handoff
Free → $16/mo (Professional)
Industry standard. Unmatched ecosystem — plugins, design systems, component libraries, MCP server integration, and the only tool every dev and designer already knows. Start free; upgrade to Professional when AI credit limits bite. Non-negotiable as the primary tool.
Tool 2 · Rapid UI Generation
Galileo AI
Prompt → Polished UI Mockup → Figma Export
$16/mo (Standard, annual)
When Cricket gets a brief like "design a task tracking dashboard," Galileo generates a production-quality UI mockup in seconds — then exports directly to Figma for polish. Converts 8 hours of blank-artboard work to 30 minutes of refinement. Huge productivity win.
Tool 3 · Asset & Image Generation
Ideogram
Image Gen, Text-in-Images, Visual Assets via API
Free → $20/mo (Plus)
Best-in-class for text-in-image (logos, social graphics, labeled icons). Has a real API for programmatic generation — critical for agent workflows. Generous free tier (~40 images/week) is enough to start. Plus plan unlocks 8,000 images/month at $20/mo.
Figma AI (Professional)$16/mo
Galileo AI (Standard)$16/mo
Ideogram (Plus)$20/mo
Total estimated monthly cost~$52/mo

Why Not the Others?

  • Midjourney — Best art, but no real API and public images on lower plans. Skip for agent workflows.
  • OpenAI gpt-image-1 — Strong API alternative to Ideogram. If K2S0 team is already deep in OpenAI tooling, swap in for Ideogram. API per-image ~$0.04 medium quality.
  • Canva — Better for marketing/social at scale. Add Pro ($15/mo) if Cricket handles lots of marketing materials. Otherwise covered by Figma.
  • Adobe Firefly — Worth it only if Adobe CC is already in the stack. Commercial-safe but less creative; API is enterprise-only.
  • Uizard / Visily — Good budget wireframe tools, outclassed by Figma+Galileo combo.
  • Looka — Useful one-off for generating a brand kit for a new project (~$65 one-time). Not a recurring tool.
